hd breakpoint

type
mixin only
use-name
scut-hd-bp
arguments
  1. $ratio [ default: 1.3 ]

    A minimum resolution.

This simple mixin provides simple means to write effective resolution-based @media-queries.

Example

SCSS
.example-hd-bp {
  color: gray;
  @include scut-hd-bp(2.0) {
    color: blue;
  }
}
Result

If you are using a retina screen, this paragraph will be blue. Otherwise, you will see a sad gray.

References

hide text

type
mixin and placeholder
use-name
scut-hide-text

Include text on the page while hiding it from view. Most commonly used to provide text (semantic, accessible, SEO-friendly, etc.) that will be (visually) replaced with an image.

Example

HTML
<p>The following block is hiding text <span class="example-hide-text">This text is hidden</span></p>
SCSS
.example-hide-text {
  @extend %scut-hide-text;
  background: blue;
  display: inline-block;
  height: 1em;
  vertical-align: middle;
  width: 1em;
}
Result

The following block is hiding text This text is hidden
