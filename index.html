general

hd breakpoint

type

mixin only

use-name

scut-hd-bp

arguments

  1. $ratio [ default: 1.3 ]

This simple mixin provides simple means to write effective resolution-based @media-queries.

Example Usage

SCSS
.example-hd-bp {
  color: gray;
  @include scut-hd-bp(2.0) {
    color: blue;
  }
}
Result

If you are using a retina screen, this paragraph will be blue. Otherwise, you will see a sad gray.
