To release:

- Add notes to changelog.
- Change version number in Gruntfile.
- Run `grunt version`.
- Build and push a new gem.
- Delete the gem.
- Commit and push.
- Create a tag: `git tag -a vx.x.x -m "[message]"` . ([See these docs](http://git-scm.com/book/en/Git-Basics-Tagging).)
- Push the tag.
- Update gh-pages.
- Update Codepen.
- Check.
