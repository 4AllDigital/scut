color swap

view source

Shorthand for setting two color-states, with the option of adding a transition.

use-name
scut-color-swap
type
mixin only
arguments
  1. $off

    Color(s) for the "off" state. A list of two values will designate color and background-color. One value alone will be interpreted as color — unless the $bg argument, below, is set to true, in which case it will be background-color.

  2. $on

    Color(s) for the "on" state (:hover and :focus). One or two values, interpreted the same as $off, above.

  3. $duration
    default
    0

    The transition-duration between $off and $on states.

  4. $bg
    default
    false

    Set to true if you want to pass a single color value for the $off and $on arguments and you want that value interpreted as background-color. (By default, a single value will be intepreted as color.)

The "on" state subsumes :hover and :focus.

example

SCSS

.eg-color-swap {
  &.m-1 {
    @include scut-color-swap($eg-dark, $eg-light, 1s);
  }
  &.m-2 {
    @include scut-color-swap($eg-light $eg-dark, $eg-dark $eg-light);
  }
  &.m-3 {
    color: white;
    @include scut-color-swap($eg-dark, $eg-light, 0.5s, true);
  }
}

compiled CSS

.eg-color-swap.m-1 {
  color: #002834;
  transition-property: color;
  transition-duration: 1s;
}
.eg-color-swap.m-1:hover, .eg-color-swap.m-1:focus {
  color: #9ae9ff;
}
.eg-color-swap.m-2 {
  color: #9ae9ff;
  background-color: #002834;
}
.eg-color-swap.m-2:hover, .eg-color-swap.m-2:focus {
  color: #002834;
  background-color: #9ae9ff;
}
.eg-color-swap.m-3 {
  color: white;
  background-color: #002834;
  transition-property: background-color;
  transition-duration: 0.5s;
}
.eg-color-swap.m-3:hover, .eg-color-swap.m-3:focus {
  background-color: #9ae9ff;
}
