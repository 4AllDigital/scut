bookends

view source

Create and style pseudo-elements on both sides of an element.

use-name
scut-bookends
type
mixin only
arguments
  1. $space
    default
    0.5em

    Spacing to separate the bookends from the element's content

  2. $content
    default
    ""

    One or two content values for the pseudo-elements. Use a list to pass different content values to the left (:before) and right (:after), respectively; or pass one value to both.

@content

A content block passed to this mixin will be applied to both :before and :after pseudo-elements.

A common typographical usage is to place decorative ornaments on either side of some heading. But the mixin can prove useful for anything that involves pseudo-elements on both sides. Absolutely anything.

Pass false as both arguments if you want to modify bookend pseudo-elements that have already been created. See example variation 3.

example

SCSS

.eg-bookends {
  &.m-1 {
    @include scut-bookends($content: "\2014");
  }
  &.m-2 {
    @include scut-bookends(0, "\201c" "\201d");
  }
  &.m-3 {
    font-size: 2em;
    @include scut-bookends(0.25em, "\00ab" "\00bb") {
      color: $eg-light;
    }
    &:hover,
    &:focus {
      color: $eg-light;
      @include scut-bookends(false, false) {
        color: $eg-dark;
      }
    }
  }
  &.m-4 {
    background-color: $eg-dark;
    color: white;
    text-align: center;
    @include scut-bookends {
      height: 2em;
      width: 2em;
      background-color: $eg-light;
      vertical-align: middle;
    }
  }
}
