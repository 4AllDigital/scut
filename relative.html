Scut Sass Utilities for the Front-End Laborer

positioning: relative

use-name
scut-relative
type
mixin only
arguments
  1. $coordinates [  default: n n n n ]

Relatively position an element and assign its coordinates, all in one line of SCSS.

-

Example

SCSS

.eg-relative {
  &.m-1 {
    @include scut-relative(0.2em 0.2em n n);
  }
  &.m-2 {
    @include scut-relative(a a 1em n);
  }
  &.m-3 {
    @include scut-relative(n n -0.5em 0.2em);
  }
}

Compiled CSS

.eg-relative.m-1 {
  position: relative;
  top: 0.2em;
  right: 0.2em;
}
.eg-relative.m-2 {
  position: relative;
  top: auto;
  right: auto;
  bottom: 1em;
}
.eg-relative.m-3 {
  position: relative;
  bottom: -0.5em;
  left: 0.2em;
}
